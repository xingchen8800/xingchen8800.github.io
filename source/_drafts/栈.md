# 栈

## T1. Text Reverse(hdu 1062)

### 题目描述
反转字符串

### 样例

### 样例输入 #1
```
olleh !dlrow
```

#### 样例输出 #1
```
hello world!
```

### 题解
用getchar()输入字符，用栈(先进后出)反转即可

#### AC代码
```c++
#include <iostream>
#include <stack>
using namespace std;

stack<char> s;

int main() {
    while(true) {
        char ch = getchar();
        if(ch == '\n' || ch == EOF || ch == ' '){
            while(!s.empty()) { //栈不空
                cout << s.top();
                s.pop();
            }
            if(ch == '\n' || ch == EOF) break;
            cout << " ";
        } else s.push(ch);
    }
    return 0;
}
```

### 手写栈
```c++
struct stack {
    int a[101];
    int _top = 0;  //栈顶指针
    void push(int x) { a[++_top] = x; }
    int top() { return a[_top]; }
    void pop() { _top--; }
    int empty() { return _top == 0; }
}
```
