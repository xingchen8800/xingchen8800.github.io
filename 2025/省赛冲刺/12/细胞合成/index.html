题目描述

小Z是某医科大学生物系的一名大学生，小Z的导师是小Z的导师。小Z的导师正在研究两种不同的细胞 细胞和 细胞。小Z的导师通过实验发现了三种有趣的酶，他将三种操作命名为!酶，@酶 和 #酶。将酶放在两个细胞之间就会出现有趣的反应。比如在两个 细胞之间加入!酶，就会将两个 细胞融合成 个 细胞，在一个 细胞和一个 细胞之间加入!酶，就会融合成 细胞。完整的效果如下表所示：

酶 A 运算 A B 运算 B A 运算 B B 运算 A
! A ! A = A B ! B = B A ! B = B B ! A = B
**@** A @ A = A B @ B = B A @ B = A B @ A = A
# A # A = B B # B = B A # B = A B # A = A

小Z的导师给小Z出了一道难题，小Z的导师准备了 个酶，并将它们排成一条线，然后小Z的导师让小Z在每个酶的两边放上 细胞或者 细胞（两个相邻的酶之间只需要放一个细胞），也就是说小Z需要放入 个细胞，生成一个反应链。小Z的导师让小Z算一算，他有多少种不同的放法( 个细胞中只要有一个不同就认为是一种不同的放法），可以使得反应链结束后生成一个 细胞。酶的反应是有优先级的，优先级如下：

  1. 括号内的酶先反应。

  2. #的优先级高于!，!号的优先级高于@的优先级。

  3. 相同优先级的酶，按照从左往右的顺序依次发生反应。

例如：A ! B # A @ (B @ A)，反应过程如下：

  1. 括号内的B @ A先反应，生成一个 ，反应链变为：A ! B # A @ A；

  2. B # A 发生反应，生成一个 ，反应链变为：A ! A @ A；

  3. A ! A 发生反应，生成一个 ，反应链变为 A @ A;

  4. A @ A 发生反应，生成一个 ，反应结束。最终生成了一个 细胞。
